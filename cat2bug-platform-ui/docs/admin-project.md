# 项目管理

## 概述

项目管理是管理员用于查看、管理系统中所有项目的功能模块。管理员可以跨团队查看和管理所有项目，进行统一的项目监控和维护。

## 功能说明

### 项目列表

项目列表展示了系统中所有项目的信息，包括：

- **项目名称**：项目的显示名称
- **所属团队**：项目所属的团队
- **项目描述**：项目的简要说明
- **项目负责人**：项目的主要负责人
- **创建时间**：项目的创建日期
- **状态**：项目的当前状态（进行中/已完成/已归档）
- **成员数量**：项目的成员总数

### 查看项目详情

管理员可以查看任何项目的详细信息：

1. 在项目列表中找到目标项目
2. 点击项目名称或"详情"按钮
3. 查看项目的完整信息，包括：
   - 基本信息
   - 成员列表
   - 缺陷统计
   - 测试用例统计
   - 项目进度

### 编辑项目

管理员可以编辑任何项目的信息：

1. 在项目列表中找到目标项目
2. 点击"编辑"按钮
3. 修改项目信息
4. 点击"确定"保存更改

### 删除项目

管理员可以删除不再需要的项目：

1. 在项目列表中找到目标项目
2. 点击"删除"按钮
3. 确认删除操作

> **警告**：删除项目会同时删除项目下的所有数据，包括缺陷、测试用例、测试计划、文档等，此操作不可恢复。

### 项目转移

管理员可以将项目从一个团队转移到另一个团队：

1. 选择需要转移的项目
2. 点击"转移"按钮
3. 选择目标团队
4. 确认转移操作

### 项目归档

管理员可以归档已完成的项目：

1. 选择需要归档的项目
2. 点击"归档"按钮
3. 确认归档操作

归档后的项目：
- 不会出现在常规项目列表中
- 数据仍然保留，可以查看
- 成员无法继续编辑项目内容
- 可以随时恢复为活跃状态

### 搜索和筛选

管理员可以通过以下条件快速查找项目：

- 项目名称关键字搜索
- 按团队筛选
- 按状态筛选
- 按创建时间筛选

## 权限说明

只有系统管理员（admin 角色）才能访问跨团队的项目管理功能。

## 常见问题

**Q: 删除项目后能否恢复？**  
A: 删除项目后数据无法恢复，建议使用归档功能代替删除。

**Q: 归档项目和删除项目有什么区别？**  
A: 归档项目保留所有数据但限制编辑，删除项目会永久删除所有数据。

**Q: 项目转移会影响项目数据吗？**  
A: 项目转移只改变项目所属团队，不会影响项目内的任何数据。

**Q: 如何恢复归档的项目？**  
A: 在项目列表中筛选"已归档"状态，找到目标项目后点击"恢复"按钮。
