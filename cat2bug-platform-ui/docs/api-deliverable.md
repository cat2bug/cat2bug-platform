# 交付物接口

## 查看交付物树形列表

**方法**：GET

**路径**：`/api/deliverable`

### Request 请求参数

| 参数名 | 类型 | 是否必填 | 说明 |
|--------|------|----------|------|
| deliverablePid | 长整型 | 否 | 交付物父ID，如不填，查询项目根节点的交付物列表 |

### Response 返回值

**返回主体说明**

| 参数名 | 类型 | 是否必返 | 说明 |
|--------|------|----------|------|
| total | 长整型 | 是 | 交付物总数 |
| rows | 数组 | 是 | 交付物树形数组 |
| code | 字符型 | 是 | 返回码，正常返回200 |
| msg | 字符型 | 是 | 接口返回消息 |

**rows 数组中交付物对象的格式说明**

| 参数名 | 类型 | 是否必返 | 说明 |
|--------|------|----------|------|
| deliverableId | 长整型 | 是 | 交付物ID |
| deliverablePid | 长整型 | 否 | 交付物的父ID |
| deliverableName | 字符型 | 是 | 交付物名称 |
| deliverablePath | 字符型 | 是 | 交付物路径 |
| childrenCount | 整型 | 是 | 子项数量 |
| remark | 字符型 | 否 | 备注 |

## 查看交付物详情

**方法**：GET

**路径**：`/api/deliverable/{交付物ID}`

### Request 请求参数

无

### Response 返回值

| 参数名 | 类型 | 是否必返 | 说明 |
|--------|------|----------|------|
| data | 对象 | 是 | 交付物对象 |
| code | 字符型 | 是 | 返回码，正常返回200 |
| msg | 字符型 | 是 | 接口返回消息 |

**data 对象结构说明**

| 参数名 | 类型 | 是否必返 | 说明 |
|--------|------|----------|------|
| deliverableId | 长整型 | 是 | 交付物ID |
| deliverablePid | 长整型 | 否 | 交付物的父ID |
| deliverableName | 字符型 | 是 | 交付物名称 |
| deliverablePath | 字符型 | 是 | 交付物路径 |
| childrenCount | 整型 | 是 | 子项数量 |
| remark | 字符型 | 否 | 备注 |

## 创建交付物

**方法**：POST

**路径**：`/api/deliverable`

### Request 请求参数

| 参数名 | 类型 | 是否必填 | 说明 |
|--------|------|----------|------|
| deliverablePid | 长整型 | 否 | 交付物父ID，如不填，创建在项目根节点下 |
| deliverableName | 字符型 | 是 | 交付物名称 |
| remark | 字符型 | 否 | 备注 |

### Response 返回值

| 参数名 | 类型 | 是否必返 | 说明 |
|--------|------|----------|------|
| data | 对象 | 是 | 创建的交付物对象 |
| code | 字符型 | 是 | 返回码，正常返回200 |
| msg | 字符型 | 是 | 接口返回消息 |

**data 对象结构说明**

| 参数名 | 类型 | 是否必返 | 说明 |
|--------|------|----------|------|
| deliverableId | 长整型 | 是 | 交付物ID |
| deliverablePid | 长整型 | 否 | 交付物的父ID |
| deliverableName | 字符型 | 是 | 交付物名称 |
| deliverablePath | 字符型 | 是 | 交付物路径 |
| childrenCount | 整型 | 是 | 子项数量 |
| remark | 字符型 | 否 | 备注 |
