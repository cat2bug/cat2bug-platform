# 文件接口

## 上传图片

**方法**：POST

**路径**：`/api/upload/image`

### Request 请求参数

| 参数名 | 类型 | 是否必填 | 说明 |
|--------|------|----------|------|
| file | 文件 | 是 | 图片文件（multipart/form-data） |

### Response 返回值

| 参数名 | 类型 | 是否必返 | 说明 |
|--------|------|----------|------|
| data | 对象 | 是 | 上传结果对象 |
| code | 字符型 | 是 | 返回码，正常返回200 |
| msg | 字符型 | 是 | 接口返回消息 |

**data 对象结构说明**

| 参数名 | 类型 | 是否必返 | 说明 |
|--------|------|----------|------|
| fileName | 字符型 | 是 | 文件名称 |
| url | 字符型 | 是 | 文件访问URL |
| newFileName | 字符型 | 是 | 新文件名（系统生成） |
| originalFilename | 字符型 | 是 | 原始文件名 |
