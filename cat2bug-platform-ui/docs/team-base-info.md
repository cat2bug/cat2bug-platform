# 基本信息

## 概述

基本信息是团队管理员用于查看和编辑团队基础信息的功能。通过基本信息设置，可以修改团队名称、描述、图标等内容。

![团队设置页面](./img/screenshots/team-setting/01-team-setting-overview.png)

## 功能说明

### 查看团队信息

在基本信息页面可以查看团队的所有基础信息：

- **团队名称**：团队的显示名称
- **团队描述**：团队的详细说明
- **团队图标**：团队的标识图标
- **创建时间**：团队的创建日期
- **团队负责人**：团队的主要负责人
- **成员数量**：团队当前的成员总数
- **项目数量**：团队下的项目总数

### 编辑团队信息

**可编辑内容：**
- 团队名称
- 团队描述
- 团队图标
- 团队负责人

**操作步骤：**
1. 点击"基本信息"卡片
2. 点击"编辑"按钮
3. 修改团队名称、描述等信息
4. 上传新的团队图标（可选）
5. 选择新的团队负责人（可选）
6. 点击"保存"完成修改

### 团队图标

**支持的图标格式：**
- JPG/JPEG
- PNG
- GIF
- 建议尺寸：200x200 像素
- 最大文件大小：2MB

**上传步骤：**
1. 点击团队图标区域
2. 选择本地图片文件
3. 裁剪图片（如需要）
4. 点击"确定"上传

### 转移团队负责人

团队负责人是团队的主要管理者，拥有最高权限。

**操作步骤：**
1. 在基本信息编辑页面
2. 点击"团队负责人"下拉框
3. 选择新的负责人（必须是团队成员）
4. 点击"保存"完成转移

> **注意**：转移负责人后，原负责人仍保留团队管理员权限，但新负责人将成为团队的主要联系人。

## 权限说明

只有团队管理员才能编辑团队基本信息。

## 常见问题

**Q: 修改团队名称会影响已有数据吗？**  
A: 不会。修改团队名称只是更改显示名称，不会影响团队下的任何数据。

**Q: 团队图标有什么用？**  
A: 团队图标用于在团队列表中快速识别团队，也会显示在团队相关的通知和报告中。

**Q: 如何转移团队负责人？**  
A: 在"基本信息"中可以修改团队负责人，选择新的负责人后保存即可。

**Q: 团队负责人和团队管理员有什么区别？**  
A: 团队负责人是团队的主要联系人，通常是团队的创建者或主要管理者；团队管理员是拥有管理权限的角色，一个团队可以有多个管理员。

**Q: 可以不设置团队负责人吗？**  
A: 不可以。每个团队必须有一个负责人，这是团队管理的基本要求。
