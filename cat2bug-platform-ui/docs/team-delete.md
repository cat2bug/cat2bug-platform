# 删除团队

## 概述

删除团队功能用于永久删除当前团队及其所有数据。这是一个不可逆的操作，只有团队管理员才能执行。

![团队设置页面](./img/screenshots/team-setting/01-team-setting-overview.png)

## 功能说明

### 删除团队

永久删除当前团队及其所有数据。

> **警告**：删除团队是不可逆操作，将永久删除团队下的所有数据，包括项目、缺陷、用例、文档等。

**操作步骤：**
1. 点击"删除团队"按钮
2. 系统会弹出确认对话框
3. 仔细阅读删除警告信息
4. 输入团队名称进行二次确认
5. 点击"确定"完成删除

### 删除前注意事项

在删除团队之前，请务必确认以下事项：

- **数据备份**：确保团队下没有重要的未备份数据
- **成员通知**：通知所有团队成员即将删除团队
- **替代方案**：考虑是否可以通过禁用团队来代替删除
- **项目影响**：删除后，团队下的所有项目也会被删除
- **成员数据**：成员创建的所有数据（缺陷、用例、文档等）都会被删除
- **无法恢复**：删除操作无法撤销，数据无法恢复

### 删除团队的影响

删除团队后会产生以下影响：

**被删除的内容：**
- 团队基本信息
- 团队下的所有项目
- 所有项目中的缺陷
- 所有项目中的测试用例
- 所有项目中的测试计划
- 所有项目中的文档
- 所有项目中的交付物
- 所有项目中的报告
- 团队成员关系（成员账号不会被删除）

**不受影响的内容：**
- 团队成员的系统账号
- 成员在其他团队中的数据
- 系统级别的配置和设置

### 替代方案

如果不确定是否要永久删除团队，可以考虑以下替代方案：

**1. 禁用团队**
- 暂时限制团队访问
- 数据完全保留
- 可以随时恢复

**2. 归档项目**
- 将不活跃的项目归档
- 保留历史数据
- 减少活跃项目数量

**3. 转移负责人**
- 将团队转交给其他管理员
- 保持团队运作
- 自己退出团队

## 权限说明

只有团队管理员才能删除团队。

## 常见问题

**Q: 删除团队后能恢复吗？**  
A: 不能。删除团队是永久性操作，请在删除前确保已备份重要数据。

**Q: 删除团队会影响团队成员的账号吗？**  
A: 不会。删除团队只是删除团队数据，不会影响成员的系统账号，成员仍可以加入其他团队。

**Q: 如果误删了团队怎么办？**  
A: 删除操作无法撤销。建议在删除前仔细确认，并做好数据备份。

**Q: 删除团队需要多长时间？**  
A: 通常在几秒到几分钟内完成，具体取决于团队数据量的大小。

**Q: 删除团队后，成员会收到通知吗？**  
A: 系统不会自动发送通知，建议在删除前手动通知所有团队成员。

**Q: 可以只删除团队下的某个项目吗？**  
A: 可以。在项目设置中可以单独删除项目，不需要删除整个团队。

**Q: 删除团队前需要先移除所有成员吗？**  
A: 不需要。删除团队时会自动解除所有成员关系。
