# 团队信息

## 概述

团队信息是团队管理员用于管理团队基础配置的功能模块。通过团队信息设置，可以修改团队基本信息和删除团队。

![团队设置页面](./img/screenshots/team-setting/01-team-setting-overview.png)

## 功能说明

### 基本信息

查看和编辑团队的基本信息。

**包含内容：**
- **团队名称**：团队的显示名称
- **团队描述**：团队的详细说明
- **团队图标**：团队的标识图标
- **创建时间**：团队的创建日期
- **团队负责人**：团队的主要负责人

**操作步骤：**
1. 点击"基本信息"
2. 修改团队名称、描述等信息
3. 点击"保存"完成修改

### 删除团队

永久删除当前团队及其所有数据。

> **警告**：删除团队是不可逆操作，将永久删除团队下的所有数据，包括项目、缺陷、用例、文档等。

**操作步骤：**
1. 点击"删除团队"
2. 确认删除操作
3. 输入团队名称进行二次确认
4. 点击"确定"完成删除

**删除前注意事项：**
- 确保团队下没有重要的未备份数据
- 通知所有团队成员即将删除团队
- 考虑是否可以通过禁用团队来代替删除
- 删除后，团队下的所有项目也会被删除

## 权限说明

只有团队管理员才能访问和修改团队信息设置。

## 常见问题

**Q: 删除团队后能恢复吗？**  
A: 不能。删除团队是永久性操作，请在删除前确保已备份重要数据。

**Q: 修改团队名称会影响已有数据吗？**  
A: 不会。修改团队名称只是更改显示名称，不会影响团队下的任何数据。

**Q: 删除团队会影响团队成员的账号吗？**  
A: 不会。删除团队只是删除团队数据，不会影响成员的系统账号，成员仍可以加入其他团队。

**Q: 如何转移团队负责人？**  
A: 在"基本信息"中可以修改团队负责人，选择新的负责人后保存即可。
