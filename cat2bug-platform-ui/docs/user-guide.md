# 用户指南

## 缺陷管理

### 创建缺陷
- 填写缺陷标题和描述
- 选择缺陷类型（功能、UI、性能等）
- 设置优先级
- 指定处理人
- 上传相关截图或附件

### 更新缺陷状态
- 待处理
- 处理中
- 已解决
- 已关闭
- 重新打开

### 缺陷评论
- 在缺陷详情页添加评论
- @提及团队成员
- 上传补充材料

## 测试用例管理

### 创建测试用例
1. 进入测试用例模块
2. 点击"新增用例"
3. 填写用例信息
4. 设置前置条件和预期结果

### 执行测试用例
1. 选择要执行的用例
2. 记录执行结果
3. 如发现问题，可直接创建缺陷

## 报告功能

### 生成测试报告
- 选择报告类型
- 设置时间范围
- 导出 PDF 或 Excel 格式

### 查看统计数据
- 缺陷趋势图
- 成员工作量统计
- 项目进度概览
