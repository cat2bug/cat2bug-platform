# 成员管理

## 概述

成员管理是管理员用于管理系统中所有用户的功能模块。管理员可以查看、创建、编辑用户信息，分配角色和权限，以及管理用户的账号状态。

## 功能说明

### 成员列表

成员列表展示了系统中所有注册用户的信息，包括：

- **用户名**：用户的登录账号
- **姓名**：用户的真实姓名
- **邮箱**：用户的电子邮箱地址
- **手机号**：用户的联系电话
- **角色**：用户拥有的角色列表
- **所属团队**：用户加入的团队数量
- **状态**：用户账号的状态（正常/禁用/锁定）
- **最后登录时间**：用户最近一次登录的时间
- **创建时间**：用户账号的创建日期

### 创建用户

管理员可以手动创建新用户：

1. 点击"新增用户"按钮
2. 填写用户信息：
   - 用户名（必填，唯一）
   - 姓名（必填）
   - 邮箱（必填）
   - 手机号
   - 初始密码（必填）
3. 分配角色（可选）
4. 点击"确定"完成创建

> **提示**：创建用户后，建议通知用户初始密码，并要求用户首次登录后修改密码。

### 编辑用户

管理员可以编辑用户的基本信息：

1. 在成员列表中找到目标用户
2. 点击"编辑"按钮
3. 修改用户信息：
   - 姓名
   - 邮箱
   - 手机号
   - 角色分配
4. 点击"确定"保存更改

> **注意**：用户名创建后不能修改。

### 重置密码

管理员可以为用户重置密码：

1. 在成员列表中找到目标用户
2. 点击"重置密码"按钮
3. 输入新密码或使用系统生成的随机密码
4. 点击"确定"完成重置

> **安全提示**：重置密码后，请通过安全渠道通知用户新密码。

### 角色分配

管理员可以为用户分配或移除角色：

1. 在成员列表中找到目标用户
2. 点击"分配角色"按钮
3. 勾选或取消勾选角色
4. 点击"确定"保存更改

用户可以同时拥有多个角色，最终权限是所有角色权限的并集。

### 用户状态管理

管理员可以管理用户账号的状态：

#### 禁用用户
- 禁用后用户无法登录系统
- 用户数据和权限配置保留
- 可以随时恢复为正常状态

操作步骤：
1. 选择目标用户
2. 点击"禁用"按钮
3. 确认操作

#### 启用用户
- 恢复被禁用用户的登录权限
- 用户可以正常登录和使用系统

操作步骤：
1. 筛选"已禁用"状态的用户
2. 选择目标用户
3. 点击"启用"按钮

#### 解锁用户
- 当用户多次输入错误密码导致账号被锁定时使用
- 解锁后用户可以重新尝试登录

操作步骤：
1. 筛选"已锁定"状态的用户
2. 选择目标用户
3. 点击"解锁"按钮

### 删除用户

管理员可以删除不再需要的用户账号：

1. 在成员列表中找到目标用户
2. 点击"删除"按钮
3. 确认删除操作

> **警告**：
> - 删除用户会同时删除用户的个人数据
> - 用户创建的内容（如缺陷、测试用例）不会被删除，但创建人信息会保留
> - 删除操作不可恢复，建议使用禁用功能代替删除

### 查看用户详情

管理员可以查看用户的详细信息：

1. 点击用户名或"详情"按钮
2. 查看用户的完整信息：
   - 基本信息
   - 角色和权限
   - 所属团队列表
   - 参与的项目列表
   - 登录历史
   - 操作日志

### 搜索和筛选

管理员可以通过以下条件快速查找用户：

- 用户名/姓名关键字搜索
- 按角色筛选
- 按状态筛选（正常/禁用/锁定）
- 按团队筛选
- 按注册时间筛选
- 按最后登录时间筛选

### 批量操作

管理员可以对多个用户执行批量操作：

1. 在成员列表中勾选多个用户
2. 选择批量操作：
   - 批量分配角色
   - 批量禁用
   - 批量启用
   - 批量删除
3. 确认操作

### 导出用户列表

管理员可以导出用户列表：

1. 设置筛选条件（可选）
2. 点击"导出"按钮
3. 选择导出格式（Excel/CSV）
4. 下载导出文件

导出内容包括：
- 用户基本信息
- 角色信息
- 账号状态
- 注册时间
- 最后登录时间

### 用户活跃度统计

管理员可以查看用户活跃度统计：

- 活跃用户数（最近7天/30天登录）
- 新增用户数（最近7天/30天）
- 长期未登录用户列表
- 用户登录趋势图

## 权限说明

只有系统管理员（admin 角色）才能访问成员管理功能。

## 安全建议

1. **定期审查**：定期审查用户列表，及时禁用或删除离职人员账号
2. **密码策略**：要求用户使用强密码，定期更换密码
3. **最小权限**：遵循最小权限原则，只授予必要的角色和权限
4. **登录监控**：关注异常登录行为，如异地登录、频繁失败等
5. **账号回收**：对长期未使用的账号进行清理或禁用

## 常见问题

**Q: 如何批量导入用户？**  
A: 可以使用系统提供的批量导入功能，下载模板文件，填写用户信息后上传。

**Q: 用户忘记密码怎么办？**  
A: 用户可以通过登录页面的"忘记密码"功能自助重置，或联系管理员重置密码。

**Q: 禁用用户和删除用户有什么区别？**  
A: 禁用用户保留所有数据和配置，可以恢复；删除用户会永久删除用户数据，不可恢复。

**Q: 如何查看某个用户的操作历史？**  
A: 在用户详情页面可以查看该用户的操作日志，包括登录记录和关键操作记录。

**Q: 用户账号被锁定的原因是什么？**  
A: 通常是因为连续多次输入错误密码（默认5次），系统会自动锁定账号30分钟，管理员可以手动解锁。

**Q: 如何限制用户只能访问特定团队或项目？**  
A: 通过角色权限配置和团队/项目成员管理来实现，用户只能访问其加入的团队和项目。

**Q: 可以设置用户账号的有效期吗？**  
A: 系统支持设置账号有效期，到期后账号会自动禁用，需要管理员手动续期。
