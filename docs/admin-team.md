# 团队管理

## 概述

团队管理是管理员用于创建、编辑和管理团队的功能模块。管理员可以通过此功能对系统中的所有团队进行统一管理。

## 功能说明

### 团队列表

团队列表展示了系统中所有已创建的团队信息，包括：

- **团队名称**：团队的显示名称
- **团队描述**：团队的简要说明
- **创建时间**：团队的创建日期
- **成员数量**：团队当前的成员总数
- **项目数量**：团队下的项目总数
- **状态**：团队的启用/禁用状态

### 创建团队

管理员可以创建新的团队：

1. 点击"新增"按钮
2. 填写团队信息：
   - 团队名称（必填）
   - 团队描述
   - 团队负责人
3. 点击"确定"完成创建

### 编辑团队

管理员可以编辑现有团队的信息：

1. 在团队列表中找到目标团队
2. 点击"编辑"按钮
3. 修改团队信息
4. 点击"确定"保存更改

### 删除团队

管理员可以删除不再需要的团队：

1. 在团队列表中找到目标团队
2. 点击"删除"按钮
3. 确认删除操作

> **注意**：删除团队前，请确保团队下没有活跃的项目和数据，删除操作不可恢复。

### 团队状态管理

管理员可以启用或禁用团队：

- **启用**：团队成员可以正常访问和使用团队资源
- **禁用**：团队成员将无法访问团队资源，但数据不会被删除

## 权限说明

只有系统管理员（admin 角色）才能访问团队管理功能。

## 常见问题

**Q: 删除团队后数据能否恢复？**  
A: 删除团队后数据无法恢复，建议在删除前先备份重要数据。

**Q: 禁用团队和删除团队有什么区别？**  
A: 禁用团队只是暂时限制访问，数据仍然保留；删除团队会永久删除所有相关数据。

**Q: 一个团队最多可以有多少成员？**  
A: 系统对团队成员数量没有限制。
