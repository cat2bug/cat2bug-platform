# 手工测试：Cat2BugTable 表头拖序与调宽

## 缺陷列表（表格视图）

1. 打开缺陷列表，hover 任意数据列表头（非操作列）。
2. 应出现左侧六点拖序手柄（pin 左侧）与右侧竖条调宽手柄；pin 常显。
3. 未 hover 与 hover 时，pin、标题、排序箭头水平位置应一致（无跳动）。
4. 拖序手柄拖动列 → 顺序变化，刷新后保持。
5. 调宽竖条拖动 → 列宽变化，刷新后保持；极窄时不低于约 60px。
6. 点击 pin 一次 → 左固定状态切换。
7. 「显示字段」显隐仍正常。
8. 切换暗色主题，手柄可见可点。

## 其他 Cat2BugTable 页（smoke）

- 用例列表、计划列表、文档、报告：hover 见手柄，拖序/调宽一次即可。
- 计划抽屉内 CaseList / DefectList：同上。

## 必填列

- 缺陷名称、处理人：标题仍为红色，手柄与 pin 正常。
