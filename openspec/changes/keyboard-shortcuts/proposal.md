## Why

平台目前完全依赖鼠标操作，缺乏键盘高效导航能力。需要一套**类 VIM 的键盘操作系统**，让用户不用鼠标即可在模块间跳转并执行常用动作，提升熟练用户效率。

关键约束：`Cmd/Ctrl + 字母` 组合在 Mac/Windows 浏览器中大量被系统保留且**网页无法拦截**（如 `Cmd+W` 关标签、`Cmd+Q` 退出、`Cmd+T` 新标签、`Cmd+N` 新窗口、`Cmd+V` 粘贴）。因此采用 **「引导键 + 字母序列」**（GitHub / Linear / VIM 同款）模型彻底规避冲突，且无需区分操作系统。

## What Changes

- **快捷键引擎（前端全局）**
  - 引导键状态机：`g` 打开导航面板、`空格` 打开当前页动作面板
  - 焦点守卫：输入框 / 文本域 / contenteditable / Excel 编辑态 / 弹框内一律禁用拦截
  - 纯前端，零后端改动；配置存 `localStorage`
- **居中命令面板（独立悬浮）**
  - 屏幕居中的浮层，列出当前上下文可用项，每项**右下角显示字母**
  - 二级面板：下拉类入口（国际化、个人中心、主题）第一次字母打开子面板，第二次字母触发具体项
  - **模糊搜索**：面板顶部搜索框按标题过滤，字母直达与模糊搜索并存
  - 支持键盘选择、`Esc` 关闭、鼠标点击兜底
- **导航映射（从实时菜单派生）**
  - 覆盖左侧菜单全部页面：缺陷 / 用例 / 测试计划 / 交付物 / 仪表盘 / 报告 / 项目文档 / 项目设置 / 团队设置 / 项目管理
  - 覆盖顶部工具：通知 / 系统文档 / 国际化（下拉）/ 个人中心（下拉）/ 主题（下拉）/ 官网 / 源码(Git)
- **布局级导航浮层（系统级）**
  - 按住 **Shift+Cmd/Ctrl**：在左侧菜单、顶部工具栏、侧栏折叠按钮显示字母徽标（与 `g` 导航共用字母表）
  - 与仅按 **Cmd/Ctrl** 的页面级/表单级浮层互斥
- **缺陷页动作映射**
  - 双通道：`空格` 命令面板 或 按住 `Cmd/Ctrl`（无 Shift）直接显示浮层徽标
  - 默认字母：L 切换项目、E 新建菜单、S 查询、J 切换 Tab、I 统计模版、G 统计模块导航（有模块时）、O 切换视图、B/P 翻页
  - 表格/Excel 可见行动态徽标（按住 `Cmd/Ctrl`）跳转对应缺陷
  - 空格 / `Cmd/Ctrl` 页面动作**仅**在缺陷列表一级界面、无抽屉/对话框/下拉浮层时生效
- **统计模版页动作**
  - P/G/H 进入预览区 / 个人模版 / 团队模版导航；方向键网格导航与跨区上下传递；Esc 退出导航或返回
- **缺陷表单键盘集成**（新建/编辑抽屉）
  - `Cmd/Ctrl+Enter` 保存、`Esc` 关闭（未保存确认）；关闭按钮无徽标，保存按钮仅在按住 `Cmd/Ctrl` 时显示 `↵`
  - 按住 `Cmd/Ctrl` 显示字段字母并跳转；字段快捷键**仅分配给当前可视区域**内表单项
  - `A/C/V/X/Z` 永不分配给字段；Tab 方向、`↑+Tab` 反向；成员/交付物/开关/上传单 Tab 停靠点
  - 字段跳转后使用全站统一 `:focus` 焦点环，不再使用临时荧光动画
- **缺陷工具弹框键盘集成**（指派/修复/驳回/通过/打开/关闭等 `el-dialog`）
  - 复用抽屉同一套：`Cmd/Ctrl+Enter` 提交、`Esc`/取消/关闭前未保存确认
  - 打开后自动聚焦首字段；按住 `Cmd/Ctrl` 显示表单字段字母徽标
- **处理缺陷抽屉键盘徽标**（`HandleDefect`）
  - 按住 `Cmd/Ctrl`：右上角工具按钮显示字母（G 指派、F 修复…）；风琴组标题数字优先（视口内动态分配）
  - `Cmd/Ctrl+↑/↓` 滚动抽屉内容区；松开修饰键立即隐藏徽标
- **统计模块配置弹框**（报时提醒、我的人生等）
  - 与缺陷工具弹框同套关闭确认 + `Cmd/Ctrl+Enter`/`Esc`/字段徽标
  - 报时提醒表格：`Cmd/Ctrl+数字` 跳行、`Cmd/Ctrl++` 新增行、`Del` 删行；时间控件失焦关面板、确定/回车后回焦、`↑/↓` 重开下拉
- **统计区 G 导航增强**
  - 进入模块导航后 `Enter`/空格触发模块主点击（筛选、打开配置弹框等），与鼠标点击等效
- **「键盘设置」配置页**
  - 入口置于顶部「个人中心」头像下拉
  - 列出全部快捷键、可编辑、实时冲突检测、恢复默认；偏好持久化到 `localStorage`

## Capabilities

### New Capabilities

- `keyboard-shortcut-engine`：全局键盘引擎（引导键状态机、焦点守卫、动作派发、页面级注册 API）
- `keyboard-command-palette`：居中悬浮命令面板（导航 / 动作 / 下拉子面板，字母角标，二次按键触发）
- `keyboard-navigation-map`：从侧边栏菜单与顶部工具栏派生的导航快捷键集合与默认字母
- `keyboard-defect-actions`：缺陷页动作快捷键集合（含 `Cmd/Ctrl` 浮层徽标与统计区 G 导航）
- `keyboard-statistic-template-actions`：统计模版页三区导航与跨区焦点传递
- `keyboard-settings-page`：键盘设置页、localStorage 持久化、冲突检测、恢复默认
- `keyboard-form-integration`：缺陷抽屉表单组合键、Tab 方向、组合组件键盘与焦点环
- `keyboard-tool-dialog-actions`：缺陷工具栏操作弹框（指派/修复等）键盘与未保存关闭确认
- `keyboard-handle-defect-actions`：处理缺陷抽屉工具栏与风琴组快捷键徽标
- `keyboard-statistic-dialog-actions`：统计模块配置弹框（报时提醒表格、时间控件等）
- `keyboard-layout-nav-hints`：Shift+Cmd 布局级导航浮层（侧栏菜单、顶栏、折叠按钮）

### Modified Capabilities

- （无）不改动现有路由、菜单、导航栏既有行为；仅在「个人中心」下拉新增一个入口项

## Impact

- **模块**：仅 `cat2bug-platform-ui`
- **新增目录**：`src/plugins/shortcut/`、`src/components/Shortcut/`、`src/views/member/keyboard/`
- **触点**：`main.js`（注册插件）、`layout/index.vue`（挂载引擎与面板）、`layout/components/Navbar.vue`（新增「键盘设置」入口）、`router/index.js`（新增设置页路由）、缺陷页（声明动作面板）、i18n 7 语言文案
- **依赖**：复用 `Sidebar` 的 `sidebarRouters`（菜单树）、`Navbar` 顶部项、现有 `$router`、`localStorage`（同 `themeMode` 模式）
- **Non-Goals（首期）**
  - 后端持久化 / 跨设备同步（仅 localStorage）
  - 缺陷页以外的页面级动作面板（首期仅做缺陷页，引擎支持后续页面扩展）
  - 序列键宏录制、多键组合（仅「引导键 + 单字母 / 单数字」两级）
  - 移动端 / 触屏适配

## 产品决策（探索阶段已确认）

1. **采用引导键序列**：导航 `g`、动作 `空格`；规避浏览器/OS 冲突，无需区分系统
2. **配置仅存前端 localStorage**
3. **导航目标全部对应左侧菜单页面 + 顶部工具栏**（含主题/官网/Git 三项）
4. **下拉入口走二级居中面板，二次按键触发**（国际化、个人中心、主题均走二级面板）
5. **提示为屏幕居中的独立悬浮面板**，菜单项右下角显示字母
6. **输入态全面禁用**快捷键拦截
7. 模糊项已澄清：交付物、测试计划、用例、项目管理均为**独立页面/列表页**
8. **命令面板提供模糊搜索**（与字母直达并存）
9. **主题走二级面板**（浅色 / 暗色），不再直接切换
