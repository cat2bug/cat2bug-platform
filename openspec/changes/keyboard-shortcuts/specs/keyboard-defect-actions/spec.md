## ADDED Requirements

### Requirement: 缺陷页动作面板

缺陷页 SHALL 在激活时注册其动作面板内容，并由动作引导键（默认 `空格`）打开。面板 MUST 仅在缺陷页激活时可触发，离开缺陷页 MUST 注销。

动作引导键 MUST 仅在缺陷列表**一级界面**、且页面上不存在可见弹框/抽屉/下拉浮层等遮挡层时生效（见 `keyboard-shortcut-engine` 遮挡层守卫）。

#### Scenario: 非缺陷页不展示缺陷动作

- **WHEN** 用户在非缺陷页按下动作引导键
- **THEN** 不展示缺陷页动作

#### Scenario: 弹框打开时不展示缺陷动作

- **WHEN** 用户在缺陷页但新建/编辑抽屉或任意对话框已打开，按下动作引导键
- **THEN** 不打开页面动作面板

### Requirement: Cmd/Ctrl 页面动作浮层

缺陷页 SHALL 混入 `page-action-hints`：按住 `Cmd/Ctrl` 时，在工具栏控件右下角显示与动作面板一致的字母浮层徽标；保持修饰键再按字母 MUST 直接执行对应动作，无需先按空格。

遮挡层守卫与动作引导键相同：有抽屉/对话框/下拉浮层时不显示徽标、不响应字母。

#### Scenario: 按住 Cmd 显示工具栏徽标

- **WHEN** 用户在缺陷列表一级界面按住 `Cmd/Ctrl`
- **THEN** 工具栏可见控件显示对应动作字母徽标

#### Scenario: Cmd+字母直接执行

- **WHEN** 用户保持 `Cmd/Ctrl` 并按下已映射字母（如 `S`）
- **THEN** 执行对应动作（如聚焦查询区）

### Requirement: 缺陷页动作默认键

系统 MUST 为缺陷页提供以下默认动作字母（用户可在键盘设置中修改）：

| 键 | 动作 |
|---|---|
| L | 切换项目 |
| E | 新建菜单（新建/导入/导出下拉） |
| S | 查询（聚焦查询区） |
| J | 切换 Tab |
| I | 统计模版（跳转配置页） |
| G | 统计模块导航（**仅统计区有模块时**） |
| O | 切换视图 |
| B | 上一页 |
| P | 下一页 |

#### Scenario: 打开新建菜单

- **WHEN** 用户在缺陷页按 `空格` 后按 `E`，或按住 `Cmd/Ctrl` 后按 `E`
- **THEN** 展开新建/导入/导出工具栏下拉

#### Scenario: 翻页

- **WHEN** 用户在缺陷页按 `空格` 后按 `P`，或按住 `Cmd/Ctrl` 后按 `P`
- **THEN** 列表翻到下一页

### Requirement: 统计模块导航（G）

当缺陷列表顶部统计区**存在至少一个统计模块**时，系统 MUST 注册 `statisticNav`（默认字母 `G`）并在统计区右下角显示徽标。统计区**无模块**时 MUST NOT 注册、显示或响应 `G`。

进入统计模块导航后：

- `←` / `→`：在模块间移动焦点
- `Delete` / `Backspace`：移除当前聚焦模块
- `Esc`：退出统计模块导航

#### Scenario: 无模块时不显示 G

- **WHEN** 统计区没有任何统计模块
- **THEN** 不显示 `G` 徽标，按 `G` 无效果

#### Scenario: 有模块时进入导航

- **WHEN** 统计区有模块且用户按 `Cmd/Ctrl+G` 或 `空格` 后 `G`
- **THEN** 进入统计模块键盘导航并聚焦第一个模块

### Requirement: 可见行动态徽标

缺陷页在表格或 Excel 视图下，按住 `Cmd/Ctrl` 时 MUST 为当前视口内可见行分配动态字母（`1`–`9` 优先，字母池补位，与工具栏已占用字母去重）。再按对应字母 MUST 打开该行缺陷。

#### Scenario: 跳转可见行缺陷

- **WHEN** 用户按住 `Cmd/Ctrl` 看到某行显示字母 `3` 并按下 `3`
- **THEN** 打开该行对应缺陷

### Requirement: 查询聚焦

缺陷页动作「查询」触发时，系统 MUST 将焦点移动到缺陷查询区并进入查询键盘导航；聚焦后引擎须遵循输入态守卫不再拦截后续字符。

#### Scenario: 聚焦查询区后正常输入

- **WHEN** 用户在缺陷页触发查询动作（`S`）
- **THEN** 焦点移到查询区，随后键入的字符进入查询控件而非触发快捷键
