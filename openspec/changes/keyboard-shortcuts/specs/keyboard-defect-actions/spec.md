## ADDED Requirements

### Requirement: 缺陷页动作面板

缺陷页 SHALL 在激活时注册其动作面板内容，并由动作引导键（默认 `空格`）打开。面板 MUST 仅在缺陷页激活时可触发，离开缺陷页 MUST 注销。

动作引导键 MUST 仅在缺陷列表**一级界面**、且页面上不存在可见弹框/抽屉/下拉浮层等遮挡层时生效（见 `keyboard-shortcut-engine` 遮挡层守卫）。

#### Scenario: 非缺陷页不展示缺陷动作

- **WHEN** 用户在非缺陷页按下动作引导键
- **THEN** 不展示缺陷页动作

#### Scenario: 弹框打开时不展示缺陷动作

- **WHEN** 用户在缺陷页但新建/编辑抽屉或任意对话框已打开，按下动作引导键
- **THEN** 不打开页面动作面板

### Requirement: 缺陷页动作默认键

系统 MUST 为缺陷页提供以下默认动作字母（用户可改）：新建缺陷 `N`、导出缺陷 `E`、导入弹框 `I`、查询(聚焦搜索) `Q`、切换 Tab `J`、统计模版 `V`、上一页 `B`、下一页 `P`。

#### Scenario: 新建缺陷

- **WHEN** 用户在缺陷页按 `空格` 后按 `N`
- **THEN** 触发新建缺陷入口

#### Scenario: 打开导入弹框

- **WHEN** 用户在缺陷页按 `空格` 后按 `I`
- **THEN** 打开导入缺陷对话框

#### Scenario: 翻页

- **WHEN** 用户在缺陷页按 `空格` 后按 `P`
- **THEN** 列表翻到下一页

### Requirement: 数字键跳转置顶缺陷

缺陷页动作面板打开时，按下数字键 `1`–`9` MUST 跳转/定位到当前列表中置顶的第 N 条缺陷。

#### Scenario: 跳到第 3 条

- **WHEN** 用户在缺陷页按 `空格` 后按 `3`
- **THEN** 定位到列表中第 3 条缺陷

### Requirement: 查询聚焦

缺陷页动作「查询」触发时，系统 MUST 将焦点移动到缺陷搜索输入框；聚焦后引擎须遵循输入态守卫不再拦截后续字符。

#### Scenario: 聚焦搜索框后正常输入

- **WHEN** 用户在缺陷页按 `空格` 后按 `Q`
- **THEN** 焦点移到搜索框，随后键入的字符进入搜索框而非触发快捷键
