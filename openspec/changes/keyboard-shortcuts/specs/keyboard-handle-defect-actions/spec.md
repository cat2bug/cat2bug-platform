## ADDED Requirements

### Requirement: 处理缺陷抽屉工具栏徽标

处理缺陷抽屉（`HandleDefect`）在按住 `Cmd/Ctrl` 时 MUST 在右上角可见工具按钮上显示字母徽标，并保持修饰键再按字母等效点击对应按钮。

默认首选字母：

| 操作 | 字母 |
|---|---|
| 指派 | G |
| 修复 | F |
| 驳回 | J |
| 通过 | P |
| 打开 | O |
| 关闭 | B |
| 编辑 | E |
| 删除 | D |
| 恢复 | U |

仅当该抽屉为栈顶可见实例时激活，避免多层抽屉串台。

#### Scenario: Cmd+G 打开指派

- **WHEN** 用户在处理缺陷抽屉内按住 `Cmd/Ctrl` 并按下 `G`
- **THEN** 等效点击指派按钮

### Requirement: 处理缺陷风琴组徽标

处理缺陷抽屉内自定义属性风琴组标题 SHALL 在按住 `Cmd/Ctrl` 时为视口内可见组分配数字优先键位（`1`–`9`、`0`，用尽后字母池），再按对应键展开该风箱。

#### Scenario: 数字键展开风箱

- **WHEN** 用户按住 `Cmd/Ctrl` 看到某风箱标题显示 `2` 并按下 `2`
- **THEN** 展开该自定义属性风箱

### Requirement: 处理缺陷抽屉滚动

按住 `Cmd/Ctrl` 时 `↑`/`↓` MUST 滚动抽屉内容区（约 40% 视口高度），滚动后刷新徽标映射。

#### Scenario: 滚动后徽标更新

- **WHEN** 用户按住 `Cmd/Ctrl` 并按下 `↓` 使风箱滚出/滚入视口
- **THEN** 风琴组数字/字母徽标随视口刷新
