# 管理员指南

## 系统配置

### 用户管理
- 创建和管理用户账号
- 分配角色和权限
- 重置用户密码
- 启用/禁用用户账号

### 角色权限管理
- 创建自定义角色
- 配置角色权限
- 数据权限设置（全部、自定义、部门、仅本人）

### 团队管理
- 创建团队
- 添加团队成员
- 设置团队管理员
- 配置团队权限

## 项目配置

### 项目设置
- 项目基本信息
- 项目成员管理
- 缺陷类型配置
- 优先级配置
- 状态流转规则

### 通知设置
- 邮件通知配置
- 系统消息推送
- WebSocket 实时通知
- 通知规则设置

## 系统监控

### 性能监控
- 查看系统运行状态
- 监控数据库连接
- 查看缓存使用情况
- 分析慢查询日志

### 操作日志
- 查看用户操作记录
- 审计关键操作
- 导出日志数据

## 数据管理

### 数据备份
- 定期备份数据库
- 导出项目数据
- 备份附件文件

### 数据导入导出
- 批量导入用户
- 导出缺陷数据
- 导出测试用例
