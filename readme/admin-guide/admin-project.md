# 项目管理 [/team/project-list](/team/project-list)

## 概述

项目管理是管理员用于查看和管理系统中所有项目的功能模块。管理员可以跨团队查看所有项目，并对项目进行禁用或解锁操作。

![项目管理界面](../images/admin-guide/04-project-manage.png)

## 功能说明

### 搜索项目

管理员可以通过多种条件快速查找项目：

1. **按团队名称搜索**：在团队名称搜索框中输入关键字
2. **按项目名称搜索**：在项目名称搜索框中输入关键字
3. 系统会自动筛选匹配的项目

### 项目列表

项目列表展示了系统中所有项目的信息，包括：

- **项目名称**：项目的显示名称
- **所属团队**：项目所属的团队
- **项目描述**：项目的简要说明
- **创建时间**：项目的创建日期
- **状态**：项目的启用/禁用状态
- **成员数量**：项目的成员总数

### 禁用项目

管理员可以禁用项目：

1. 在项目列表中找到目标项目
2. 点击"禁用"按钮
3. 确认禁用操作

禁用后的项目：
- 项目成员将无法访问项目资源
- 项目数据仍然保留
- 可以随时解锁恢复

### 解锁项目

管理员可以解锁被禁用的项目：

1. 在项目列表中找到被禁用的项目
2. 点击"解锁"按钮
3. 项目恢复正常状态

## 权限说明

只有系统管理员（admin 角色）才能访问跨团队的项目管理功能。

## 常见问题

**Q: 禁用项目会删除项目数据吗？**  
A: 不会。禁用项目只是暂时限制访问，所有数据都会保留。

**Q: 如何查看某个团队下的所有项目？**  
A: 在团队名称搜索框中输入团队名称，系统会筛选出该团队的所有项目。

**Q: 禁用项目后，项目成员还能看到项目吗？**  
A: 禁用后，项目成员将无法访问该项目的任何内容。
