# 缺陷接口文档

## 查看缺陷列表

方法：GET

路径：/api/defect

Request请求参数：

| 参数名 | 类型 | 是否必填 | 说明 |
|:---|:---:|:----:|:---|
| pageNum | 整型 | 否 | 页码，默认为1 |
| pageSize | 整型 | 否 | 每页数量，默认为10 |
| defectName | 字符型 | 否 | 缺陷名称 |
| defectState | 整型 | 否 | 缺陷状态：0-待处理；1-处理中；2-已处理；3-已关闭；4-已拒绝 |
| defectType | 整型 | 否 | 缺陷类型：0-需求；1-缺陷；2-任务；3-其它 |
| defectLevel | 整型 | 否 | 缺陷等级：0-建议；1-普通；2-严重；3-致命 |
| handleBy | 长整型 | 否 | 处理人ID |
| createBy | 长整型 | 否 | 创建人ID |

Response返回值：

* 返回主体说明

| 参数名 | 类型 | 是否必返 | 说明 |
|:---|:---:|:----:|:---|
| total | 长整型 | 是 | 缺陷总数 |
| rows | 数组 | 是 | 缺陷数组 |
| code | 字符型 | 是 | 返回码，正常返回200 |
| msg | 字符型 | 是 | 接口返回消息 |

* rows数组中缺陷对象的格式说明

| 参数名 | 类型 | 是否必返 | 说明 |
|:---|:---:|:----:|:---|
| defectId | 长整型 | 是 | 缺陷ID |
| defectNum | 字符型 | 是 | 缺陷编号 |
| defectName | 字符型 | 是 | 缺陷名称 |
| defectDescribe | 字符型 | 否 | 缺陷描述 |
| defectState | 整型 | 是 | 缺陷状态：0-待处理；1-处理中；2-已处理；3-已关闭；4-已拒绝 |
| defectType | 整型 | 是 | 缺陷类型：0-需求；1-缺陷；2-任务；3-其它 |
| defectLevel | 整型 | 是 | 缺陷等级：0-建议；1-普通；2-严重；3-致命 |
| handleBy | 长整型 | 否 | 处理人ID |
| createBy | 长整型 | 是 | 创建人ID |
| createTime | 日期型 | 是 | 创建时间 |
| updateTime | 日期型 | 是 | 更新时间 |
| creator | 对象 | 是 | 创建人信息 |
| handler | 对象 | 否 | 处理人信息 |

* creator/handler对象结构说明

| 参数名 | 类型 | 是否必返 | 说明 |
|:---|:---:|:----:|:---|
| memberId | 长整型 | 是 | 成员ID |
| memberAccount | 字符型 | 是 | 成员账号 |
| memberName | 字符型 | 是 | 成员名称 |
| phoneNumber | 字符型 | 是 | 手机号 |
| email | 字符型 | 否 | 电子邮箱 |

## 创建缺陷

方法：POST

路径：/api/defect

Request请求参数：

| 参数名 | 类型 | 是否必填 | 说明 |
|:---|:---:|:----:|:---|
| defectName | 字符型 | 是 | 缺陷名称 |
| defectDescribe | 字符型 | 否 | 缺陷描述 |
| defectType | 整型 | 是 | 缺陷类型：0-需求；1-缺陷；2-任务；3-其它 |
| defectLevel | 整型 | 是 | 缺陷等级：0-建议；1-普通；2-严重；3-致命 |
| handleBy | 长整型 | 否 | 处理人ID |
| moduleId | 长整型 | 否 | 模块ID |
| annexList | 数组 | 否 | 附件列表 |

* annexList数组中附件对象的格式说明

| 参数名 | 类型 | 是否必填 | 说明 |
|:---|:---:|:----:|:---|
| annexName | 字符型 | 是 | 附件名称 |
| annexUrl | 字符型 | 是 | 附件URL |
| annexType | 整型 | 是 | 附件类型：0-图片；1-视频；2-音频；3-文档；4-其它 |

Response返回值：

| 参数名 | 类型 | 是否必返 | 说明 |
|:---|:---:|:----:|:---|
| data | 对象 | 是 | 创建的缺陷对象 |
| code | 字符型 | 是 | 返回码，正常返回200 |
| msg | 字符型 | 是 | 接口返回消息 |

* data对象结构说明（同查看缺陷列表中的缺陷对象）
