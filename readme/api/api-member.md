# 成员接口

## 查看项目成员列表

**方法**：GET

**路径**：`/api/member`

### Request 请求参数

| 参数名 | 类型 | 是否必填 | 说明 |
|--------|------|----------|------|
| pageNum | 整型 | 否 | 页码，默认为1 |
| pageSize | 整型 | 否 | 每页数量，默认为10 |
| memberName | 字符型 | 否 | 成员名称 |
| memberAccount | 字符型 | 否 | 成员账号 |

### Response 返回值

**返回主体说明**

| 参数名 | 类型 | 是否必返 | 说明 |
|--------|------|----------|------|
| total | 长整型 | 是 | 成员总数 |
| rows | 数组 | 是 | 成员数组 |
| code | 字符型 | 是 | 返回码，正常返回200 |
| msg | 字符型 | 是 | 接口返回消息 |

**rows 数组中成员对象的格式说明**

| 参数名 | 类型 | 是否必返 | 说明 |
|--------|------|----------|------|
| memberId | 长整型 | 是 | 成员ID |
| memberAccount | 字符型 | 是 | 成员账号 |
| memberName | 字符型 | 是 | 成员名称 |
| phoneNumber | 字符型 | 是 | 手机号 |
| email | 字符型 | 否 | 电子邮箱 |
| sex | 字符型 | 否 | 性别：0-男；1-女；2-未知 |
| avatar | 字符型 | 否 | 头像URL |
| createTime | 日期型 | 是 | 创建时间 |
