# 键盘快捷键 [键盘设置](/member/keyboard)

## 概述

Cat2Bug 提供一套**键盘优先**的操作方式，让您在熟悉键位后，尽量少用鼠标即可完成页面跳转和常用动作。快捷键采用「引导键 + 单字母」模型，避免与浏览器保留的 `Ctrl/Cmd + 字母` 组合冲突。

系统中有三种交互层次：

| 层次 | 操作方式 | 适用场景 |
|------|----------|----------|
| 引导键 | 按 `g` 或 **空格**，再按字母 | 跨页面导航、当前页主要动作 |
| 页面级 | **按住 ⌘（Mac）或 Ctrl（Windows/Linux）** | 工具栏按钮、表单字段、表格可见行 |
| 布局级 | **按住 ⇧⌘ 或 Shift+Ctrl** | 左侧菜单、顶栏工具、侧栏折叠、团队选择 |

> **说明**：下文统一写作 **⌘/Ctrl** 表示 Mac 的 Command 键或 Windows/Linux 的 Ctrl 键；**⇧⌘/Shift+Ctrl** 表示同时按住 Shift 与 ⌘/Ctrl。

默认键位可在 [键盘设置](/member/keyboard) 页查看和修改。若本文表格与设置页不一致，**以设置页当前显示为准**。

## 通用操作

### 引导键

| 按键 | 作用 |
|------|------|
| `g` | 打开**导航命令面板**，列出侧栏菜单与顶栏入口；2 秒内再按目标字母执行 |
| **空格** | 打开**当前页动作面板**（仅部分页面支持，如缺陷列表、系统文档页） |
| `Esc` | 关闭命令面板；在下拉菜单中关闭菜单；在文档页可逐级退出导航或返回上一页 |

面板打开后，可在顶部搜索框输入关键字过滤条目，仍可用字母直达。

### 修饰键浮层

| 按键 | 作用 |
|------|------|
| 按住 **⌘/Ctrl** | 在当前页面控件右下角显示字母徽标，继续按字母即可触发 |
| 按住 **⇧⌘/Shift+Ctrl** | 在侧栏菜单、顶栏图标、侧栏折叠按钮上显示字母徽标 |
| 松开修饰键 | 徽标立即消失 |

通过 ⇧⌘/Shift+Ctrl 打开的下拉菜单（团队、语言、个人中心等）中，可用 **↑ / ↓** 切换选项，**Enter** 确认，**Esc** 关闭。

### 何时快捷键不会触发

以下情况系统**不会**拦截按键，以便正常输入或操作：

- 焦点在输入框、文本域、下拉框或 Excel 单元格编辑态
- 正在使用输入法组合输入（IME）
- 焦点在按钮或链接上时，**空格**不会打开动作面板
- 已打开对话框、抽屉、消息框或下拉浮层（如成员选择、日期面板）时，**空格**不会打开页面动作面板
- 按住 ⌘/Ctrl 的页面徽标在弹框/抽屉打开时也会隐藏

登录页的老鼠 D-pad 操控（⌘/Ctrl + 方向键）为登录页专属装饰交互，不走全局引导键引擎。

### 布局级（⇧⌘/Shift+Ctrl）

按住 **⇧⌘/Shift+Ctrl** 时，侧栏各菜单项、顶栏图标与折叠按钮会显示与 `g` 导航相同的字母（侧栏折叠默认为 **`-`**）。**须同时按住 Shift**，与仅按 ⌘/Ctrl 的页面浮层不同；具体字母以界面徽标为准。

## 自定义快捷键

### 入口

1. 点击右上角头像
2. 选择 **键盘设置**，或按 `g` → `U` → `K`
3. 也可直接访问 [/member/keyboard](/member/keyboard)

### 可配置项

- **总开关**：关闭后全局快捷键不再拦截按键
- **引导键**：导航（默认 `g`）、动作（默认空格）
- **分组键位**：导航、各页面动作、布局级等，点击字母可重新绑定
- **冲突检测**：同组内重复字母会标红提示
- **恢复默认**：支持单项或全部恢复

### 存储说明

设置保存在**当前浏览器**的本地存储（localStorage）中，**不会**同步到其他设备或浏览器。换电脑或清空站点数据后需重新配置。

## 常见问题

**Q: 按空格没有打开动作面板？**  
A: 请确认：(1) 当前页面已支持动作面板（如缺陷列表）；(2) 未打开抽屉、对话框或下拉菜单；(3) 焦点不在输入框或按钮上；(4) 键盘设置中总开关已开启。

**Q: 按住 ⌘/Ctrl 看不到字母徽标？**  
A: 弹框或抽屉打开时页面徽标会隐藏；请先关闭浮层。Mac 请按 Command，Windows 请按 Ctrl。若仍无显示，检查键盘设置是否关闭总开关。

**Q: 在 Excel 模式编辑单元格时快捷键无效？**  
A: 这是预期行为。编辑单元格时按键留给表格输入；结束编辑后再使用快捷键。

**Q: 与浏览器快捷键冲突怎么办？**  
A: 导航与页面动作使用「引导键 + 单字母」，刻意避开 `Ctrl+W` 等浏览器组合。布局级使用 ⇧⌘/Shift+Ctrl，仍可能与少数浏览器扩展冲突，可在键盘设置中改字母。

**Q: 表格行的字母为什么和文档写的不一样？**  
A: 行号徽标按**当前可见行**动态分配，不是固定映射。以按住 ⌘/Ctrl 时界面上显示的字母为准。

**Q: 修改键位后文档还有效吗？**  
A: 本文描述**默认**键位。自定义后以 [键盘设置](/member/keyboard) 页显示为准。
