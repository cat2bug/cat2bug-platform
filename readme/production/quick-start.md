# 快速开始

本指南将帮助您快速上手 Cat2Bug 系统，完成从登录到创建缺陷的完整流程。

## 整体流程

使用 Cat2Bug 进行缺陷管理的基本流程如下：

```
┌─────────────┐
│  登录系统    │
└──────┬──────┘
       ▼
┌─────────────┐
│  创建团队    │
└──────┬──────┘
       ▼
┌─────────────┐
│ 添加团队成员 │
└──────┬──────┘
       ▼
┌─────────────┐
│  创建项目    │
└──────┬──────┘
       ▼
┌─────────────┐
│添加项目成员  │
└──────┬──────┘
       ▼
┌─────────────┐
│创建测试用例  │
│  (可选)     │
└──────┬──────┘
       ▼
┌─────────────┐
│ 创建测试计划 │
│  (可选)     │
└──────┬──────┘
       ▼
┌─────────────┐
│  创建缺陷    │
└──────┬──────┘
       ▼
┌─────────────┐
│  团队协作    │
└─────────────┘
```

**流程说明：**

1. **登录系统** - 使用账号密码登录系统
2. **创建团队** - 创建一个团队作为协作单位
3. **添加团队成员** - 邀请或创建团队成员（必须步骤，项目成员只能从团队成员中选择）
4. **创建项目** - 在团队下创建项目
5. **添加项目成员** - 从团队成员中选择项目参与者
6. **创建测试用例**（可选）- 编写测试用例，规范测试流程
7. **创建测试计划**（可选）- 制定测试计划，组织测试活动
8. **创建缺陷** - 在项目中记录和管理缺陷
9. **团队协作** - 分配、处理和跟踪缺陷

下面将详细介绍每个步骤的操作方法。

## 登录系统

1. 访问系统地址：http://localhost:2222
2. 输入用户名和密码
3. 点击登录按钮

## 创建团队

在创建项目之前，需要先创建一个团队：

1. 点击左侧菜单"团队管理"
2. 点击"新增"按钮
3. 填写团队信息：
   - 团队名称
   - 团队介绍（可选）
4. 点击"确定"保存

## 添加团队成员

创建团队后，必须添加团队成员，因为项目成员只能从团队成员中选择：

1. 进入团队详情页
2. 点击"成员管理"
3. 添加成员的方式：
   - **邀请现有用户**：输入用户账号或邮箱，发送邀请
   - **创建新成员**：填写成员信息，系统自动创建账号
4. 设置成员角色和权限
5. 点击"确定"保存

> **重要提示**：项目成员必须先是团队成员，所以在创建项目前务必先添加团队成员。

## 创建项目

添加团队成员后，就可以在团队下创建项目：

1. 点击左侧菜单"项目管理"
2. 点击"新增"按钮
3. 填写项目信息：
   - 选择所属团队
   - 项目名称
   - 项目介绍（可选）
   - 项目图标（可选）
4. 点击"确定"保存

## 添加项目成员

创建项目后，需要为项目添加成员：

1. 进入项目详情页
2. 点击"项目设置" → "成员管理"
3. 从团队成员列表中选择项目参与者
4. 为每个成员分配项目角色：
   - 项目管理员
   - 开发人员
   - 测试人员
   - 观察者
5. 点击"确定"保存

> **注意**：只能从团队成员中选择项目成员，如果需要添加新成员，请先在团队中添加。

## 创建测试用例（可选）

为了规范测试流程，可以创建测试用例：

1. 进入项目详情页
2. 点击左侧菜单"测试用例"
3. 点击"新增"按钮
4. 填写用例信息：
   - 用例标题
   - 前置条件
   - 测试步骤
   - 预期结果
   - 优先级
5. 点击"保存"

## 创建测试计划（可选）

可以创建测试计划来组织测试活动：

1. 进入项目详情页
2. 点击左侧菜单"测试计划"
3. 点击"新增"按钮
4. 填写计划信息：
   - 计划名称
   - 测试范围
   - 开始/结束时间
   - 关联测试用例
   - 指派测试人员
5. 点击"保存"

## 创建缺陷

1. 进入项目详情页
2. 点击"新增缺陷"按钮
3. 填写缺陷信息
4. 上传截图（可选）
5. 点击"提交"

## 团队协作

1. 在项目中添加团队成员
2. 分配缺陷给相应成员
3. 成员可以更新缺陷状态
4. 通过评论进行沟通
