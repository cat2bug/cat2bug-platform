# 项目创建人

## 角色概述

项目创建人是创建项目的用户，拥有项目的最高权限。项目创建人负责项目的整体规划、配置和管理。

## 主要职责

- 创建和配置项目
- 管理项目成员
- 分配项目管理员
- 设置项目权限和规则
- 监督项目进展
- 删除项目

## 核心权限

### 项目管理
- ✅ 修改项目信息（名称、介绍、图标等）
- ✅ 删除项目
- ✅ 配置项目所有设置
- ✅ 设置项目可见性

### 成员管理
- ✅ 添加项目成员（从团队成员中选择）
- ✅ 移除项目成员
- ✅ 设置项目管理员
- ✅ 取消项目管理员权限
- ✅ 分配成员角色（开发、测试等）

### 功能配置
- ✅ 配置缺陷管理规则
- ✅ 设置测试用例模板
- ✅ 配置测试计划流程
- ✅ 管理项目文档
- ✅ 设置第三方集成（钉钉、飞书等）
- ✅ 配置 AI 功能

### 数据管理
- ✅ 查看所有项目数据
- ✅ 导出项目数据
- ✅ 管理项目报告
- ✅ 配置数据权限

## 操作流程

### 创建项目

1. 进入团队管理
2. 选择要创建项目的团队
3. 点击"创建项目"
4. 填写项目信息：
   - 项目名称
   - 项目介绍
   - 项目图标
5. 点击"确定"完成创建

### 添加项目成员

1. 进入项目详情页
2. 点击"项目设置" → "成员管理"
3. 从团队成员列表中选择成员
4. 分配项目角色：
   - 项目管理员
   - 开发人员
   - 测试人员
   - 观察者
5. 点击"确定"

### 设置项目管理员

1. 进入项目成员列表
2. 找到要设置的成员
3. 点击"设为管理员"
4. 确认操作

### 配置项目设置

1. 进入"项目设置"
2. 可配置以下内容：
   - **基本信息**：修改项目名称、介绍等
   - **API KEY**：管理项目 API 密钥
   - **AI 管理**：配置 AI 大模型和账号
   - **第三方应用**：集成钉钉、飞书、企业微信
   - **成员管理**：管理项目成员和权限

### 删除项目

1. 进入"项目设置" → "项目信息"
2. 滚动到底部
3. 点击"删除项目"
4. 输入密码确认
5. 确认删除

> **警告**：删除项目会永久删除所有项目数据，此操作不可恢复！

## 最佳实践

### 项目规划
- 创建项目前明确项目目标和范围
- 合理规划项目结构和模块
- 设置清晰的项目命名和描述
- 制定项目管理规范

### 成员管理
- 根据职责合理分配项目角色
- 至少设置一名项目管理员作为备份
- 定期审查项目成员和权限
- 及时移除不再参与的成员

### 权限配置
- 遵循最小权限原则
- 敏感操作设置审批流程
- 定期审查权限配置
- 记录重要权限变更

### 数据安全
- 定期备份项目数据
- 重要操作前做好确认
- 保护 API KEY 等敏感信息
- 设置合理的数据访问权限

## 常见问题

**Q: 项目创建人可以转让吗？**  
A: 目前不支持直接转让项目创建人身份，但可以设置多个项目管理员来分担管理工作。

**Q: 删除项目会影响团队吗？**  
A: 删除项目只会删除该项目的数据，不会影响团队和其他项目。

**Q: 项目创建人可以退出项目吗？**  
A: 项目创建人不能直接退出项目，如需退出，需要先删除项目或联系团队管理员。

**Q: 可以创建多少个项目？**  
A: 每个团队可以创建的项目数量由系统配置决定。

**Q: 项目成员必须是团队成员吗？**  
A: 是的，项目成员只能从所属团队的成员中选择。
