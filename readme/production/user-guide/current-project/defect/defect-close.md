# 关闭缺陷

当缺陷问题已经解决，或者当前不再考虑处理此问题，可直接将其关闭。

![关闭缺陷](../../../images/defect-close.png)

## 使用场景

- 缺陷已解决并验证通过
- 不予修复的问题
- 重复的缺陷
- 无法重现的问题
- 需求变更导致缺陷失效

## 操作步骤

### 1. 选择缺陷

在缺陷列表中选择要关闭的缺陷。

### 2. 点击关闭

点击缺陷右侧的【关闭】按钮。

### 3. 选择关闭原因

在弹出的对话框中选择关闭原因：
- **已解决** - 问题已修复并验证通过
- **不予修复** - 当前不考虑修复此问题
- **重复缺陷** - 与其他缺陷重复
- **无法重现** - 无法重现问题
- **需求变更** - 需求变更导致缺陷失效
- **其他** - 其他原因

### 4. 填写关闭说明

填写关闭说明，详细说明关闭原因。

### 5. 确认关闭

点击【确认】按钮完成关闭。

## 注意事项

> **提示：**
> 1. 关闭的缺陷可以重新开启
> 2. 关闭记录会保存在操作历史中
> 3. 建议在关闭说明中详细说明原因
> 4. 关闭后会自动通知相关人员
