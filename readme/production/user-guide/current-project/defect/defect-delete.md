# 删除缺陷

删除后的缺陷信息将无法找回，一般只有创建错误或确认无效的缺陷，才可以删除。

## 使用场景

- 创建错误的缺陷
- 测试数据需要清理
- 确认无效的缺陷记录

## 操作步骤

### 1. 点击删除

在缺陷列表或却想详情中点击【删除】按钮。

### 2. 确认删除

在弹出的确认对话框中，确认删除操作。

## 删除影响

删除缺陷会导致：
- 缺陷信息永久删除，无法恢复
- 关联的评论和附件一并删除
- 相关的操作历史记录删除
- 统计数据中不再包含此缺陷

::: tip 提示
1. 删除操作不可恢复，请谨慎操作
2. 只有缺陷创建人、测试人员、项目管理员有删除权限
3. 建议使用"关闭"而不是"删除"
4. 对于已处理的缺陷，不建议删除，以保留历史记录
:::
