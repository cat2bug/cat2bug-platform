# 文档管理 [/project/document](/project/document)

文档模块用于存储和管理项目相关文件，支持文件夹分类管理，可通过新建、更新、移动、删除等操作进行管理。

## 什么是文档管理

在项目开发和测试过程中，会产生大量的文档资料，如需求文档、设计文档、测试方案、会议纪要等。文档管理模块提供了一个集中存储和管理这些文档的地方。

**文档管理的作用：**

- 集中存储项目文档
- 支持文件夹分类管理
- 方便团队成员查阅
- 支持文档在线编辑
- 便于文档检索和下载

## 文档列表

文档列表展示了项目中的所有文档和文件夹，支持多种查看和操作方式。

![文档列表](../../../images/user-guide/current-project/document/doc-list.png)

### 列表信息

文档列表包含以下列：

- **文档/文件夹名称** - 显示文件夹图标或文档图标及名称
- **文档类型** - 显示文档的文件类型（如 docx、pdf 等），文件夹显示"文件夹"
- **更新人** - 最后更新该文档或文件夹的用户
- **更新时间** - 最后更新的时间
- **操作** - 提供移动、更新、删除等操作按钮

### 顶部操作

文档列表右上方提供以下操作按钮：

- **新建文件夹** - 创建新的文件夹用于分类管理文档
- **新建文档** - 在线创建新的文档

### 搜索功能

在列表顶部的搜索框中输入文档或文件夹名称，可快速搜索定位。

**搜索特点：**
- 支持模糊匹配
- 实时搜索
- 搜索文档名称和文件夹名称

## 文档类型

系统支持多种文档格式：

**支持的文件格式：**
- **文档类** - Word (.doc, .docx), PDF (.pdf), Excel (.xls, .xlsx), PPT (.ppt, .pptx)
- **文本类** - TXT (.txt), Markdown (.md)
- **图片类** - JPG, PNG, GIF, BMP
- **压缩包** - ZIP, RAR, 7Z
- **其他** - 根据系统配置支持更多格式
