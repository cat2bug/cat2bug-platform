# 查看文档

查看文档的详细内容，支持在线预览多种格式的文档。

## 使用场景

- 查看文档内容
- 预览文档效果
- 检查文档格式
- 阅读项目文档

## 打开文档

### 从文档列表打开

在文档列表中点击文档名称或【查看】按钮，打开文档详情页面。

### 从搜索结果打开

在搜索结果中点击文档，直接打开文档详情。

### 从通知打开

收到文档更新通知后，点击通知中的链接直接打开文档。

## 在线预览

系统支持在线预览多种格式的文档，无需下载到本地。

![文档预览](../../../images/document-view.png)

### PDF 文档预览

**预览功能：**
- 翻页浏览
- 缩放查看
- 全屏模式
- 搜索文本
- 打印文档

**操作方式：**
- 点击翻页按钮切换页面
- 使用缩放按钮调整大小
- 按 F11 进入全屏模式
- 使用搜索框查找文本

### 图片文件预览

**支持的格式：**
- JPG / JPEG
- PNG
- GIF
- BMP

**预览功能：**
- 原图查看
- 缩放查看
- 旋转图片
- 下载图片

### Markdown 文档预览

**预览特点：**
- 渲染后的效果展示
- 支持代码高亮
- 支持表格展示
- 支持图片显示
- 支持数学公式
- 支持流程图

**预览模式：**
- 阅读模式 - 只显示渲染后的内容
- 编辑模式 - 显示源码和预览

### 文本文件预览

**支持的格式：**
- TXT
- LOG
- CSV

**预览功能：**
- 语法高亮（代码文件）
- 行号显示
- 搜索文本
- 复制内容

### Office 文档预览

**支持的格式：**
- Word（.doc, .docx）
- Excel（.xls, .xlsx）
- PowerPoint（.ppt, .pptx）

**预览说明：**
- 需要配置在线预览服务
- 支持基本的格式展示
- 复杂格式可能显示不完整
- 建议下载后使用 Office 软件查看

## 文档信息

文档详情页面显示文档的详细信息。

### 基本信息

- **文档名称** - 文档的标题
- **文档类型** - 需求文档、设计文档等
- **文件大小** - 文档的大小
- **文件格式** - 文档的格式
- **创建人** - 文档上传者
- **创建时间** - 文档上传时间
- **更新时间** - 最后修改时间

### 文档描述

显示文档的详细描述信息。

### 文档标签

显示文档的标签，点击标签可以查看相同标签的其他文档。

## 文档操作

在文档详情页面可以进行多种操作。

### 下载文档

点击【下载】按钮，下载文档到本地。

**下载方式：**
- 直接下载原文件
- 保留原文件名
- 保存到浏览器默认下载目录

### 编辑文档

点击【编辑】按钮，编辑文档信息或内容。

**可编辑内容：**
- 文档标题
- 文档描述
- 文档类型
- 文档标签
- 文档内容（Markdown 文档）

### 更新文档

点击【更新】按钮，上传新版本的文档。

**更新说明：**
- 系统会保留历史版本
- 需要填写更新说明
- 更新后版本号自动增加

### 分享文档

点击【分享】按钮，生成文档分享链接。

**分享方式：**
- 复制分享链接
- 生成二维码
- 发送邮件通知

**分享权限：**
- 项目成员可以直接访问
- 外部人员需要登录后访问
- 可以设置分享有效期

### 删除文档

点击【删除】按钮，删除文档。

**删除确认：**
- 需要确认删除操作
- 删除后不可恢复
- 历史版本一并删除

## 版本历史

查看文档的历史版本。

### 查看版本历史

1. 在文档详情页面点击【版本历史】标签
2. 查看所有历史版本列表
3. 每个版本显示：
   - 版本号
   - 更新时间
   - 更新人
   - 更新说明
   - 文件大小

### 版本对比

对比不同版本的文档内容。

**对比步骤：**
1. 在版本历史中选择两个版本
2. 点击【对比】按钮
3. 查看差异内容
4. 高亮显示变更部分

**对比功能：**
- 并排对比
- 高亮显示差异
- 显示新增内容（绿色）
- 显示删除内容（红色）
- 显示修改内容（黄色）

### 版本恢复

恢复到历史版本。

**恢复步骤：**
1. 在版本历史中找到要恢复的版本
2. 点击【恢复】按钮
3. 确认恢复操作
4. 系统创建新版本，内容为历史版本的内容

**恢复说明：**
- 不会删除当前版本
- 创建新版本作为恢复结果
- 保留完整的版本历史

### 下载历史版本

可以下载任意历史版本的文档。

**下载步骤：**
1. 在版本历史中找到要下载的版本
2. 点击【下载】按钮
3. 下载该版本的文档文件

## 文档评论

可以对文档添加评论和讨论。

### 添加评论

1. 在文档详情页面底部找到评论区
2. 输入评论内容
3. 点击【发表】按钮
4. 评论会显示在文档下方

**评论功能：**
- 支持 Markdown 格式
- 支持 @提及成员
- 支持上传图片
- 支持回复评论

### 查看评论

评论按时间顺序显示：
- 最新评论在上方
- 显示评论人和时间
- 显示评论内容
- 显示回复数量

### 回复评论

1. 点击评论下方的【回复】按钮
2. 输入回复内容
3. 点击【发表】按钮
4. 回复会显示在原评论下方

### 删除评论

点击自己评论右侧的【删除】按钮可以删除评论。

**删除权限：**
- 只能删除自己的评论
- 项目管理员可以删除任何评论
- 删除评论不可恢复

## 文档导航

### 目录导航

对于 Markdown 文档，左侧显示文档目录。

**目录功能：**
- 自动提取标题生成目录
- 点击目录项跳转到对应章节
- 高亮显示当前章节
- 支持折叠/展开

### 锚点链接

文档中的标题自动生成锚点，可以通过 URL 直接定位。

**锚点格式：**
```
https://cat2bug.com/document/123#章节标题
```

### 返回顶部

点击页面右下角的【返回顶部】按钮，快速回到文档开头。

::: tip 提示
1. 支持在线预览多种格式的文档
2. Markdown 文档支持实时渲染和代码高亮
3. 可以查看文档的版本历史和对比差异
4. 支持对文档添加评论和讨论
5. 分享链接可以设置有效期和访问权限
:::
