# 交付物列表

交付物列表展示了项目中的所有交付物，以树形结构呈现交付物的层级关系。

## 使用场景

- 查看项目的交付物结构
- 快速定位某个交付物
- 了解各交付物的统计信息
- 管理交付物的层级关系

## 列表展示

交付物以树形结构展示，清晰地呈现父子关系。

![交付物列表](../../../images/deliverable-list.png)

### 列表信息

每个交付物显示以下信息：

- **交付物名称** - 交付物的名称
- **描述** - 交付物的简要说明
- **负责人** - 负责该交付物的成员
- **用例数** - 关联的测试用例数量
- **缺陷数** - 关联的缺陷数量
- **操作按钮** - 编辑、删除等操作

## 树形结构操作

### 展开/折叠

- **展开节点** - 点击节点前的 ▶ 图标展开子节点
- **折叠节点** - 点击节点前的 ▼ 图标折叠子节点
- **全部展开** - 点击工具栏的【全部展开】按钮
- **全部折叠** - 点击工具栏的【全部折叠】按钮

### 拖拽调整

可以通过拖拽方式调整交付物的层级关系：

1. 选中要移动的交付物
2. 拖拽到目标位置
3. 系统自动保存新的层级关系

## 搜索功能

在搜索框中输入关键字，可以快速找到交付物。

**搜索范围：**
- 交付物名称
- 交付物描述

**搜索特点：**
- 支持模糊匹配
- 实时搜索
- 高亮显示匹配内容

## 统计信息

### 用例统计

显示每个交付物关联的测试用例数量：

- **总用例数** - 该交付物及其子交付物的用例总数
- **各级别分布** - P0、P1、P2、P3 用例数量

### 缺陷统计

显示每个交付物的缺陷情况：

- **总缺陷数** - 该交付物及其子交付物的缺陷总数
- **各状态分布** - 处理中、待验证、已驳回、已关闭
- **各优先级分布** - 急、高、中、低

## 快速操作

### 快速创建子交付物

在交付物节点上右键，选择【添加子交付物】，可以快速创建子交付物。

### 快速查看关联数据

点击用例数或缺陷数，可以快速跳转到对应的列表页面，查看该交付物关联的用例或缺陷。

::: tip 提示
1. 交付物的统计数据包含其所有子交付物的数据
2. 拖拽调整层级时，子交付物会随父交付物一起移动
3. 搜索时会自动展开包含匹配结果的节点
4. 建议交付物层级不超过 4 层，以保持结构清晰
:::
