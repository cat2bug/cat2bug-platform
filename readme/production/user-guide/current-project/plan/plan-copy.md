# 复制计划

复制计划可以快速创建相似的测试计划，避免重复配置，提高工作效率。

## 使用场景

- 进行回归测试时，复制上一版本的测试计划
- 创建相似模块的测试计划
- 基于已有计划快速创建新的测试计划

## 操作步骤

### 1. 选择要复制的计划

在测试计划列表中，找到需要复制的测试计划。

![复制测试计划](../../../images/user-guide/current-project/plan/plan_copy.png)

### 2. 点击复制按钮

点击测试计划右侧的【复制】按钮，系统会复制计划并刷新列表。

## 注意事项

:::tip 提示：
1. 复制的计划会包含原计划的所有用例，但不会复制执行状态
2. 建议修改计划名称，避免与原计划混淆
3. 根据实际情况调整测试用例，确保计划的准确性
:::
