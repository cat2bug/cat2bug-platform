# 测试计划模板

系统提供了常用的测试计划模板，可以快速创建符合不同测试场景的测试计划。

## 使用模板创建计划

在创建测试计划时，可以选择使用预定义的模板，系统会自动填充相应的配置和用例。

![选择测试计划模板](../../../images/user-guide/current-project/plan/plan_template_select.png)

## 功能测试计划模板

适用于功能测试，全面验证系统功能是否符合需求。

### 模板内容

- **功能用例集** - 覆盖所有功能点的测试用例
- **界面测试用例** - 验证界面显示和交互
- **业务流程测试用例** - 验证完整的业务流程

### 适用场景

- 新功能开发完成后的功能验证
- 系统功能全面测试
- 验收测试前的功能确认

![功能测试计划模板](../../../images/user-guide/current-project/plan/plan_template_function.png)

## 回归测试计划模板

适用于版本迭代的回归测试，确保新版本不影响已有功能。

### 模板内容

- **核心功能用例（P0、P1）** - 最重要的核心功能用例
- **历史缺陷验证用例** - 验证历史缺陷是否复现
- **影响范围相关用例** - 本次修改可能影响的功能用例

### 适用场景

- 版本迭代后的回归测试
- 缺陷修复后的验证
- 代码重构后的功能验证

![回归测试计划模板](../../../images/user-guide/current-project/plan/plan_template_regression.png)

## 冒烟测试计划模板

适用于快速验证，确保基本功能可用。

### 模板内容

- **核心业务流程用例** - 最关键的业务流程
- **关键功能用例** - 系统的核心功能
- **基础功能用例** - 登录、权限等基础功能

### 适用场景

- 新版本部署后的快速验证
- 每日构建的基本验证
- 环境搭建后的可用性验证

![冒烟测试计划模板](../../../images/user-guide/current-project/plan/plan_template_smoke.png)

## 验收测试计划模板

适用于项目验收，确保系统满足验收标准。

### 模板内容

- **需求覆盖用例** - 覆盖所有需求的测试用例
- **验收标准验证用例** - 验证是否满足验收标准
- **用户场景测试用例** - 模拟真实用户使用场景

### 适用场景

- 项目交付前的验收测试
- 客户验收演示
- 里程碑验收

![验收测试计划模板](../../../images/user-guide/current-project/plan/plan_template_acceptance.png)

## 自定义模板

除了系统预定义的模板，还可以创建自定义模板。

### 创建自定义模板

1. 创建一个完整的测试计划
2. 点击【保存为模板】按钮
3. 填写模板名称和描述
4. 保存模板

![创建自定义模板](../../../images/user-guide/current-project/plan/plan_template_custom.png)

### 管理模板

在模板管理页面，可以查看、编辑和删除自定义模板。

![管理测试计划模板](../../../images/user-guide/current-project/plan/plan_template_manage.png)

## 使用技巧

> **提示：**
> 1. 根据测试目标选择合适的模板
> 2. 使用模板后可以根据实际情况调整用例
> 3. 将常用的测试计划保存为模板，提高效率
> 4. 定期更新模板内容，保持模板的有效性
> 5. 团队可以共享自定义模板，统一测试标准
