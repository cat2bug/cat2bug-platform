# OpenAI账号管理 [/project/openai](/project/openai)

管理项目的OpenAI账号配置。

**功能：**
- 添加OpenAI API密钥
- 配置组织ID
- 设置使用额度
- 查看使用统计

**操作步骤：**
1. 点击"OpenAI账号管理"
2. 输入API密钥
3. 配置相关参数（组织ID、使用限额等）
4. 保存配置

**配置项说明：**
- **API Key**：OpenAI提供的访问密钥
- **Organization ID**：OpenAI组织标识（可选）
- **使用限额**：设置每月最大使用额度
- **默认模型**：选择默认使用的GPT模型版本

## 权限说明

只有项目管理员才能配置OpenAI账号。

## 常见问题

**Q: 如何获取OpenAI API密钥？**  
A: 访问OpenAI官网（https://platform.openai.com），注册账号后在API Keys页面创建密钥。
