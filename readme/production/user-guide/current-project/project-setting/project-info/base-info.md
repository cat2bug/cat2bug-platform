# 基本信息 [/project/project-base-info](/project/project-base-info)

查看和编辑项目的基本信息。

## 项目信息

基本信息页面包含以下内容：

- **项目名称** - 项目的显示名称
- **项目描述** - 项目的详细说明
- **项目图标** - 项目的标识图标
- **创建时间** - 项目的创建日期
- **创建人** - 项目的创建者

## 操作步骤

1. 进入项目设置 → 项目信息 → 基本信息
2. 修改项目名称、描述或上传项目图标
3. 点击【保存】按钮完成修改

![基本信息](../../../../../images/user-guide/current-project/project-setting/project-setting-base-info.png)

::: tip 注意事项
- 只有项目创建人和项目管理员可以修改基本信息
- 修改项目名称不会影响已有数据
- 项目图标支持 JPG、PNG 格式，建议尺寸 200x200 像素
:::
