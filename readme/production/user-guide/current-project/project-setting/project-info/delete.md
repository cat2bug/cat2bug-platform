# 删除项目

永久删除当前项目及其所有数据。

## 操作步骤

1. 进入项目设置 → 项目信息 → 删除项目
2. 点击【删除项目】按钮
3. 在确认对话框中输入当前用户的密码进行二次确认
4. 点击【确定】按钮完成删除

![删除项目](../../../../../images/user-guide/current-project/project-setting/project-setting-delete.png)

## 删除影响

删除项目将永久删除以下数据：

- 项目基本信息
- 所有测试用例
- 所有缺陷数据
- 所有交付物
- 所有文档
- 所有报告
- 项目成员关系
- 项目配置信息

::: tip 警告
- 删除项目是不可逆操作，删除后无法恢复
- 删除前请确保已备份重要数据
- 只有项目创建人可以删除项目
- 删除操作需要输入当前用户密码进行二次确认
:::
