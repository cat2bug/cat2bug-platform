# 同步到云端 [/project/push](/project/push)

将本地部署的Cat2Bug-Platform项目数据同步到官网云服务。

## 功能说明

同步到云端功能可以将本地部署的项目数据同步到Cat2Bug官网云服务（https://cloud.cat2bug.com），实现数据备份和多端访问。

## 前提条件

在使用同步功能前，需要完成以下准备工作：

1. 在Cat2Bug官网云服务（https://cloud.cat2bug.com）注册账号
2. 在云端创建一个空项目
3. 在云端项目设置中申请同步密钥

## 操作步骤

1. 进入项目设置 → 项目信息 → 同步到云端
2. 输入从云端项目获取的同步密钥
3. 点击【同步】按钮
4. 等待同步完成

![同步到云端](../../../../../images/user-guide/current-project/project-setting/project-setting-sync.png)

## 同步内容

同步功能会将以下数据同步到云端：

- 项目基本信息
- 测试用例
- 缺陷数据
- 交付物信息
- 项目成员
- 其他项目配置

::: tip 注意事项
- 只有项目创建人和项目管理员可以执行同步操作
- 同步前请确保云端项目为空，避免数据冲突
- 同步过程中请勿关闭页面
- 同步密钥请妥善保管，不要泄露给他人
- 首次同步可能需要较长时间，请耐心等待
:::
