# 第三方应用 [/project/ding](/project/ding)

## 概述

第三方应用是项目管理员用于集成外部通讯和协作工具的模块。通过配置第三方应用，可以实现消息通知、数据同步等功能，提升团队协作效率。

![项目设置页面](../../../images/user-guide/current-project/project-setting/01-project-setting-overview.png)

## 功能说明

### 钉钉 [/project/ding](/project/ding)

集成钉钉，实现消息通知和数据同步。

**功能：**
- 配置钉钉机器人
- 设置消息推送规则
- 测试钉钉连接

**操作步骤：**
1. 在钉钉群中添加自定义机器人
2. 复制机器人的Webhook地址
3. 在项目设置中点击"钉钉"
4. 粘贴Webhook地址
5. 配置消息推送规则（缺陷创建、状态变更等）
6. 点击"测试"验证配置
7. 保存配置

**消息推送场景：**
- 新缺陷创建
- 缺陷状态变更
- 缺陷分配通知
- 测试计划开始/结束
- 项目重要事件

## 配置说明

### 获取钉钉机器人Webhook

1. 打开钉钉群聊
2. 点击右上角"..."，选择"群设置"
3. 点击"智能群助手"
4. 点击"添加机器人"
5. 选择"自定义"机器人
6. 设置机器人名称和安全设置
7. 复制生成的Webhook地址

### 获取飞书机器人Webhook

1. 打开飞书群聊
2. 点击右上角"..."，选择"设置"
3. 点击"群机器人"
4. 点击"添加机器人"
5. 选择"自定义机器人"
6. 设置机器人名称
7. 复制生成的Webhook地址

### 获取企业微信机器人Webhook

1. 打开企业微信群聊
2. 点击右上角"..."
3. 点击"添加群机器人"
4. 设置机器人名称
5. 复制生成的Webhook地址

## 权限说明

只有项目管理员才能配置第三方应用集成。

## 常见问题

**Q: 如何配置多个第三方应用？**  
A: 可以同时配置钉钉、飞书、企业微信，系统会向所有已配置的平台推送消息。

**Q: 消息推送失败怎么办？**  
A: 请检查Webhook地址是否正确、机器人是否被移除、网络连接是否正常。

**Q: 可以自定义消息推送内容吗？**  
A: 目前系统提供预设的消息模板，可以选择推送哪些类型的事件。

**Q: 机器人消息会@所有人吗？**  
A: 默认不会@所有人，只有在配置中明确设置的情况下才会@相关人员。

**Q: 如何停止消息推送？**  
A: 在对应的第三方应用配置中，取消勾选消息推送规则或删除Webhook地址即可。
