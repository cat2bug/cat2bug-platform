# 基本信息 [/team/setting](/team/setting)

## 概述

基本信息用于查看和编辑当前团队的核心资料，包括团队名称、团队图标和团队介绍。页面以表单形式展示，修改后点击「保存」即可生效。

## 进入方式

1. 在左侧菜单进入 **团队设置**（路径：`/team/setting`）。
2. 在 **团队信息** 卡片中点击 **基本信息** 链接，进入基本信息页面。
3. 需要返回时，点击页面左上角的 **返回** 按钮。

## 页面说明

基本信息页面标题为 **团队信息**，面包屑为：`首页 / 团队设置 / 基本信息`。

![团队基本信息页面](../../../images/user-guide/team-setting/01-team-setting-overview.png)

页面包含以下字段：

| 字段 | 是否必填 | 说明 |
|------|----------|------|
| 团队名称 | 是 | 团队在系统中的显示名称，最多 64 个字符 |
| 团队图标 | 是 | 团队标识图，用于团队列表等位置展示 |
| 团队介绍 | 否 | 团队的简要说明，最多 255 个字符，右下角显示字数统计 |

## 操作步骤

1. 按上述方式进入基本信息页面，系统自动加载当前团队数据。
2. 在 **团队名称** 输入框中修改名称。
3. 在 **团队图标** 区域上传或更换图标（见下文）。
4. 在 **团队介绍** 文本框中填写或修改介绍（可选）。
5. 点击页面底部的 **保存** 按钮；保存成功后系统会提示「保存成功」并刷新页面。

## 团队图标

- **支持格式**：png、jpg、jpeg
- **大小限制**：不超过 5MB
- **数量**：仅可上传 1 张

**上传步骤：**

1. 点击图标区域的 **+** 按钮，从本机选择图片；也可使用剪贴板按钮粘贴图片（若界面提供）。
2. 上传成功后，预览区会显示当前图标，右上角出现绿色勾选标记。
3. 需要更换时，先移除已有图标，再重新上传。
4. 修改完成后点击 **保存**。

## 权限说明

只有 **团队创建人** 和 **团队管理员** 可以进入并修改基本信息。

普通团队成员在团队设置页不会看到 **基本信息** 入口。

## 常见问题

**Q: 修改团队名称会影响已有数据吗？**  
A: 不会。修改团队名称仅更改显示名称，不影响团队下的项目、缺陷等数据。

**Q: 团队图标有什么用？**  
A: 团队图标用于在团队列表、切换团队等场景中识别团队，便于快速区分不同团队。

**Q: 团队介绍可以不填吗？**  
A: 可以。团队介绍为选填项，留空不影响保存。

**Q: 保存失败怎么办？**  
A: 请检查团队名称、团队图标是否已填写，图标格式与大小是否符合要求；若仍失败，请确认当前账号是否为团队创建人或团队管理员。

**Q: 谁可以修改基本信息？**  
A: 团队创建人和团队管理员可以修改；普通成员无此权限。

## 键盘快捷键

通用说明见 [键盘快捷键](../../../advanced/keyboard-shortcuts.md)。

| 按键 | 动作 |
|------|------|
| **Esc** | 返回团队设置（有未保存修改时会确认） |
| **B** | 同 Esc，通过页头返回 |
| **⌘/Ctrl + Enter** | 保存 |
| 按住 **⌘/Ctrl** | 显示字段字母徽标，再按字母聚焦对应表单项 |
