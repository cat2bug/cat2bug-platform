# 仪表盘 [/project/dashboard](/project/dashboard)

仪表盘是项目的数据概览页面，提供项目关键指标的可视化展示，帮助团队快速了解项目状态。

## 功能概述

仪表盘展示以下关键信息：

### 项目概况
- **项目名称**：当前项目的名称
- **项目成员数**：项目团队人数
- **创建时间**：项目创建日期
- **项目状态**：进行中/已完成/已暂停

### 缺陷统计
- **缺陷总数**：项目中的缺陷总量
- **待处理缺陷**：尚未解决的缺陷数量
- **已解决缺陷**：已修复的缺陷数量
- **缺陷趋势图**：缺陷数量随时间的变化趋势

### 测试进度
- **测试用例总数**：项目中的测试用例数量
- **已执行用例**：已完成测试的用例数量
- **测试通过率**：测试用例的通过百分比
- **测试计划进度**：当前测试计划的执行进度

### 团队活跃度
- **活跃成员**：最近活跃的团队成员
- **最新动态**：项目中的最新操作记录
- **待办事项**：分配给当前用户的待处理任务

## 数据刷新

仪表盘数据会自动刷新，也可以点击刷新按钮手动更新数据。

## 权限说明

所有项目成员都可以查看仪表盘，但看到的数据范围可能因角色权限而异。
