# API KEY [/project/project-api](/project/project-api)

管理项目的API访问密钥，用于第三方系统集成。

**功能：**
- 查看当前API密钥
- 重新生成API密钥
- 复制API密钥

> **安全提示**：API密钥具有项目访问权限，请妥善保管，不要泄露给未授权人员。

**操作步骤：**
1. 点击"API KEY"
2. 查看或复制当前密钥
3. 如需更换，点击"重新生成"

## 权限说明

只有项目管理员才能访问和管理API密钥。

## 常见问题

**Q: API密钥泄露了怎么办？**  
A: 立即在"API KEY"中点击"重新生成"，旧密钥将立即失效。
