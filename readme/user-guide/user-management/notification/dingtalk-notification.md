# 钉钉通知

## 概述

钉钉通知允许用户通过钉钉接收系统通知，支持企业个人推送和机器人群发两种方式。

## 功能特点

- **钉钉企业个人推送**：通知直接发送到个人钉钉账号
- **钉钉机器人群发**：通过钉钉机器人向群组发送通知
- **实时推送**：通知实时推送到钉钉客户端
- **移动办公**：随时随地接收通知

## 通知方式

### 企业个人推送

- 通知发送到个人钉钉账号
- 适合接收个人相关的通知
- 需要企业钉钉应用支持

### 机器人群发

- 通知发送到钉钉群组
- 适合团队协作场景
- 需要项目管理员配置钉钉机器人

## 前置条件

### 企业个人推送

- 企业已开通钉钉企业应用
- 用户已加入企业钉钉组织

### 机器人群发

- 需要项目管理员在项目设置中完成钉钉企业应用账号绑定和配置
- 需要在钉钉群中添加自定义机器人
- 需要配置机器人的 Webhook 地址

## 启用钉钉通知

1. 访问通知页面 `/notice/index`
2. 点击"通知设置"按钮
3. 切换到"接收平台"标签页
4. 勾选"钉钉"
5. 点击"保存"按钮

> **提示**：使用钉钉群发通知前，需要项目管理员在项目设置中完成钉钉企业应用账号绑定和配置。

## 通知内容

钉钉通知包含以下信息：

- 通知标题
- 通知内容
- 相关链接（可直接跳转到系统）
- 发送时间

## 最佳实践

- 根据团队使用习惯选择钉钉通知
- 重要项目启用机器人群发，确保团队成员都能收到
- 定期检查钉钉通知设置是否正常
- 测试通知是否能正常接收

## 常见问题

**Q: 为什么收不到钉钉通知？**  
A: 请检查以下几点：
- 确认通知设置中已勾选"钉钉"
- 检查项目管理员是否已配置钉钉企业应用
- 确认您已加入企业钉钉组织
- 检查钉钉客户端的通知权限设置

**Q: 钉钉机器人群发需要什么配置？**  
A: 需要项目管理员在项目设置中配置钉钉机器人的 Webhook 地址和相关参数。

**Q: 可以自定义钉钉通知的内容吗？**  
A: 钉钉通知的内容由系统自动生成，用户无法自定义。

**Q: 钉钉通知会有延迟吗？**  
A: 钉钉通知通常是实时推送的，但可能会因网络状况有轻微延迟。

**Q: 如何关闭钉钉通知？**  
A: 在通知设置的接收平台中，取消勾选"钉钉"即可。
