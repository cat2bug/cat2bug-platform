# 飞书通知

## 概述

飞书通知允许用户通过飞书接收系统通知，支持企业个人推送和机器人群发两种方式。

## 功能特点

- **飞书企业个人推送**：通知直接发送到个人飞书账号
- **飞书机器人群发**：通过飞书机器人向群组发送通知
- **实时推送**：通知实时推送到飞书客户端
- **移动办公**：随时随地接收通知

## 通知方式

### 企业个人推送

- 通知发送到个人飞书账号
- 适合接收个人相关的通知
- 需要企业飞书应用支持

### 机器人群发

- 通知发送到飞书群组
- 适合团队协作场景
- 需要项目管理员配置飞书机器人

## 前置条件

### 企业个人推送

- 企业已开通飞书企业应用
- 用户已加入企业飞书组织

### 机器人群发

- 需要项目管理员在项目设置中完成飞书企业应用账号绑定和配置
- 需要在飞书群中添加自定义机器人
- 需要配置机器人的 Webhook 地址

## 启用飞书通知

1. 访问通知页面 `/notice/index`
2. 点击"通知设置"按钮
3. 切换到"接收平台"标签页
4. 勾选"飞书"
5. 点击"保存"按钮

> **提示**：使用飞书群发通知前，需要项目管理员在项目设置中完成飞书企业应用账号绑定和配置。

## 通知内容

飞书通知包含以下信息：

- 通知标题
- 通知内容
- 相关链接（可直接跳转到系统）
- 发送时间

## 最佳实践

- 根据团队使用习惯选择飞书通知
- 重要项目启用机器人群发，确保团队成员都能收到
- 定期检查飞书通知设置是否正常
- 测试通知是否能正常接收

## 常见问题

**Q: 为什么收不到飞书通知？**  
A: 请检查以下几点：
- 确认通知设置中已勾选"飞书"
- 检查项目管理员是否已配置飞书企业应用
- 确认您已加入企业飞书组织
- 检查飞书客户端的通知权限设置

**Q: 飞书机器人群发需要什么配置？**  
A: 需要项目管理员在项目设置中配置飞书机器人的 Webhook 地址和相关参数。

**Q: 可以自定义飞书通知的内容吗？**  
A: 飞书通知的内容由系统自动生成，用户无法自定义。

**Q: 飞书通知会有延迟吗？**  
A: 飞书通知通常是实时推送的，但可能会因网络状况有轻微延迟。

**Q: 如何关闭飞书通知？**  
A: 在通知设置的接收平台中，取消勾选"飞书"即可。
