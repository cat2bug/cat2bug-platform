# 企业微信通知

## 概述

企业微信通知允许用户通过企业微信接收系统通知，支持个人推送和群发推送两种方式。

## 功能特点

- **企业微信个人推送**：通知直接发送到个人企业微信账号
- **企业微信群发推送**：通过企业微信向群组发送通知
- **实时推送**：通知实时推送到企业微信客户端
- **移动办公**：随时随地接收通知

## 通知方式

### 个人推送

- 通知发送到个人企业微信账号
- 适合接收个人相关的通知
- 需要企业微信应用支持

### 群发推送

- 通知发送到企业微信群组
- 适合团队协作场景
- 需要项目管理员配置企业微信应用

## 前置条件

### 个人推送

- 企业已开通企业微信应用
- 用户已加入企业微信组织

### 群发推送

- 需要项目管理员在项目设置中完成企业微信企业应用账号绑定和配置
- 需要配置企业微信应用的相关参数

## 启用企业微信通知

1. 访问通知页面 `/notice/index`
2. 点击"通知设置"按钮
3. 切换到"接收平台"标签页
4. 勾选"企业微信"
5. 点击"保存"按钮

> **提示**：使用企业微信群发通知前，需要项目管理员在项目设置中完成企业微信企业应用账号绑定和配置。

## 通知内容

企业微信通知包含以下信息：

- 通知标题
- 通知内容
- 相关链接（可直接跳转到系统）
- 发送时间

## 最佳实践

- 根据团队使用习惯选择企业微信通知
- 重要项目启用群发推送，确保团队成员都能收到
- 定期检查企业微信通知设置是否正常
- 测试通知是否能正常接收

## 常见问题

**Q: 为什么收不到企业微信通知？**  
A: 请检查以下几点：
- 确认通知设置中已勾选"企业微信"
- 检查项目管理员是否已配置企业微信应用
- 确认您已加入企业微信组织
- 检查企业微信客户端的通知权限设置

**Q: 企业微信群发推送需要什么配置？**  
A: 需要项目管理员在项目设置中配置企业微信应用的相关参数。

**Q: 可以自定义企业微信通知的内容吗？**  
A: 企业微信通知的内容由系统自动生成，用户无法自定义。

**Q: 企业微信通知会有延迟吗？**  
A: 企业微信通知通常是实时推送的，但可能会因网络状况有轻微延迟。

**Q: 如何关闭企业微信通知？**  
A: 在通知设置的接收平台中，取消勾选"企业微信"即可。
